# Prisma Fundamentals

## Install Prisma
npm add prisma

## Initialize Prisma
npx prisma init 

## Initialize Prisma Studio
npx prisma studio

## Retorn DB exist
npx prisma db pull

___
## Running the database tables.
npx prisma generate

## Create Migrate
npx prisma migrate dev

## Execute
npx ts-node-dev server.ts
